
# Library Management System (C++)

## 📌 Description

This is a console-based Library Management System developed using C++. It allows users to manage books efficiently with features like add, search, update, and delete.

## 🚀 Features

* Add new books
* Display all books
* Search book by ID
* Update book details
* Delete books
* File handling for data persistence

## 🛠️ Technologies Used

* C++
* OOP Concepts
* File Handling (fstream)

## ▶️ How to Run

```bash
g++ src/main.cpp src/Library.cpp -o app
./app
```
